# Changelog

## 0.1.0 - 2026-04-25

Первая портфолио-версия проекта.

Реализовано:
- интерактивный CLI для анализа вакансий;
- режимы `manual_prompt`, `manual_json_save` и `api` с fallback без API-ключа;
- анализ вакансии из `.txt` файла через команду `analyze-file`;
- Pydantic-валидация JSON-ответов;
- сохранение результатов в `output/analyses.csv`;
- просмотр сохранённых анализов с фильтрами по `decision` и `fit_score`;
- локальный поиск красных флагов и risk score с настройкой через `config/risk_weights.json`;
- экспорт одного или нескольких анализов в Markdown;
- общий Markdown-отчёт по сохранённым анализам;
- автотесты через pytest;
- GitHub Actions workflow для проверки синтаксиса и запуска тестов.
